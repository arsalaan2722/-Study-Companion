About Study Companion


Study Companion is a modern, high-productivity educational web application built to help students, researchers, and self-learners digest complex documents faster.

Equipped with instant client-side file extraction (PDF, DOCX, TXT) and powered by Groq LLaMA 3.3 70B AI, it transforms hours of reading into structured, actionable study materials within seconds.

Features include multi-style note generation, client-side PDF export, slide layout outlines, and persistent session history stored securely in your browser.
